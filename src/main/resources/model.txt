# 请求
## 方法：POST
## 接口：/
## 请求参数
|参数名称|参数类型|是否必传|默认值|参数描述|
|--|--|--|--|--|
|pageNum|String|YES||分页请求，当前页数|


## 返回数据模板
```
{
  "innerException": "",
  "status": "200",
  "message": "",
  "data": ""
}
```

## 返回说明
|参数名称|参数类型|参数描述|
|--|--|--|
|innerException|String|返回说明|
|status|String|状态|
|message|String|消息|
|data|String|数据|
